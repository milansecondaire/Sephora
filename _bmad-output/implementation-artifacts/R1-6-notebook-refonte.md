### Cluster profiling heatmap

```python
profile_df = df_customers.groupby("final_cluster")[FEATURES_CONTINUOUS].mean()
global_mean = df_customers[FEATURES_CONTINUOUS].mean()
zscore_df = (profile_df - global_mean) / df_customers[FEATURES_CONTINUOUS].std()
fig, ax = plt.subplots(figsize=(18, 8))
sns.heatmap(zscore_df.T, annot=True, fmt=".2f", cmap="RdBu_r", center=0, ax=ax)
ax.set_title("Cluster Profiles — Z-score vs. Global Mean")
fig.tight_layout(); fig.savefig("figures/cluster_profiles.png", dpi=150)
```
